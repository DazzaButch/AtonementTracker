# AtonementTracker

A high-performance icon tracker for Discipline Priests in World of Warcraft: Midnight.

## Features
- **Taint-Safe Logic:** Specifically designed to avoid UI errors in instances and raids.
- **Group Aware:** Tracks Atonement across yourself, target, focus, party, and raid members.
- **Customizable:** Use `/at` to change icon size, opacity, and font size.
- **Clean UI:** Displays a white whole-number timer and a member count.

## Commands
- `/at` - Open the configuration window.
